Travelo Social Media

The goal of this project was to design social media graphics for a travel company with a millennial demographic. The design was inspired by travel journals and blends the digital media with elements of the handmade and physical. This project includes a set of nine posts exploring five different travel locations as well as another set of nine focusing on one location.
